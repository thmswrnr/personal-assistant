# YouTube transcript

Get a YouTube video's transcript as clean plain text via a small CLI (uses `yt-dlp`
under the hood to download only the caption track — no video/audio). Then summarize
or answer the user's question from the transcript.

## Command (run via bash)

```bash
# Accepts a full URL or a bare 11-char video id; optional --lang (default en)
node /app/.pi/skills/youtube/transcript.mjs "https://www.youtube.com/watch?v=VIDEO_ID"
node /app/.pi/skills/youtube/transcript.mjs "VIDEO_ID" --lang de
```

It prints JSON:
`{id, title, uploader, durationSeconds, url, lang, autoGenerated, chars, truncated, transcript}`.

## How to use it

1. Run the command with the URL the user gave. Then do what they asked — usually
   **summarize**: lead with a one-line gist, then the key points as bullets. Keep it
   tight unless they ask for detail.
2. If `autoGenerated` is true, the captions are machine-made — they may contain small
   errors and no punctuation; summarize the meaning, don't quote verbatim as exact.
3. If `truncated` is true (long video), say so and summarize the portion you have.
4. If it errors with "no subtitles/captions available", tell the user plainly — some
   videos simply have no captions. Try `--lang` if the video is in another language.
5. **Don't invent content** — only summarize what the transcript actually contains.
